# OpenDeck

"Open Up" is a social game where players take turns drawing cards and following the challenge written or taking a drink. It is best played with 5 players.

## About

The game was built in JavaScript with just normal HTML/CSS. It features a sleek, dark-themed interface with swipe-based card interaction and three difficulty levels of cards.

## Progressive Web App (PWA)

OpenDeck is a Progressive Web App that can be installed on your device for a native app-like experience.

### Features

- Fully responsive design that works on mobile and desktop
- Swipe-based card interaction
- Three difficulty levels of cards (White/Orange/Red)
- Download counter that tracks installations
- Works offline after initial installation
- No app store required - installs directly from the web

### Installing on iOS (iPhone/iPad)

1. Open Safari browser (PWA installation works best in Safari)
2. Navigate to the OpenDeck website
3. Tap the **Share** button (the square with an arrow pointing up)
4. Scroll down and tap **"Add to Home Screen"**
5. Choose a name for the app (or use the default "OpenDeck")
6. Tap **"Add"**
7. The app will now be available on your home screen

### Installing on Android

1. Open Chrome browser
2. Navigate to the OpenDeck website
3. Tap the **three dots** menu in the upper right corner
4. Select **"Add to Home screen"**
5. Tap **"Add"**
6. The app will now be available on your home screen

### Installing on Desktop (Chrome/Firefox/Edge)

1. Open the OpenDeck website in your browser
2. Look for the install icon in the address bar (usually looks like a "+" or "install" button)
3. Click the install button
4. Confirm the installation
5. The app will be installed and can be launched from your applications menu

### Offline Functionality

OpenDeck works offline! Once installed, you can play the game even without an internet connection. All game data is stored locally on your device.

## Development

To run the development server:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
